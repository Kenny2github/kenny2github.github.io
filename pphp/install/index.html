Installation

Assuming you have pip (if you don't, tsk tsk!), type the following into your operating system's terminal:

pip install pphp

This will install PPHP and all of its non-existant dependencies.


Installation without pip

  1. Download the package from PyPI
  2. Unzip it
  3. In a terminal, cd into the directory that you unzipped it to
  4. In the terminal, type python setup.py install

Implementation

This section assumes you are using BaseHTTPServer. You will have to adapt it if you are using anything else.

  1. At the top of your file, add from pphp import do.
  2. In your do_* functions, at the end, use self.wfile.write(do(html, GET_DATA, POST_DATA, **kwargs)), where html is the contents of your file, GET_DATA is a dictionary containing the GET parameters, (if applicable) POST_DATA is a dictionary containing the POST data, and **kwargs is any further arguments your file will require to execute.

Usage

Inside your file, to execute Python code, use:

<?pphp
(python code)
?>

Some special keywords:

  • echo(s) - this is equivalent to sys.stdout.write(s), used if you need to output something without a trailing newline.
  • _GET - this is the dictionary of GET parameters that you passed to do in implementation.
  • _POST - this is the dictionary of POST data that you passed to do in implementation.
  • _EXTRAS - this is a dictionary of **kwargs that you passed to do in implementation.
  • __script__ - this is a string containing the text of the currently running script.
