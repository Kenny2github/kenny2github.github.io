Installation

Assuming you have pip (if you don't, tsk tsk!), type the following into your operating system's terminal:

pip install pphp

This will install PPHP and all of its non-existant dependencies.


Installation without pip

  1. Download the package from PyPI
  2. Unzip it
  3. In a terminal, cd into the directory that you unzipped it to
  4. In the terminal, type python setup.py install
