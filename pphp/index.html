Python PPHP: Hypertext Preprocessor

PPHP 0.1.2

Changes:

  • __dom__ is no longer a keyword and lxml is no longer a dependency - it caused trouble when scripts were inside element parameters.
  • __script__ is a new keyword that contains the text of the current running script.
  • Fixed some bugs :)

PPHP Released

Python PPHP: Hypertext Preprocessor is a spinoff of PHP, in Python - use it to execute Python scripts inside HTML, much like PHP does.

To install it, open up a command processor (cmd on Windows or Terminal on *NIX or Mac) and type:

pip install pphp

See the installation instructions for more details.
